# R-dio-Max---Hrajeme-Hity-Pro-Maxim-ln-N-ladu
Naše rádio je místem, kde se každý den setkávají lidé, hudba, informace i emoce. Vysíláme srdcem, protože věřím, že rádio má stále sílu spojovat, inspirovat a doprovázet posluchače v každé životní situaci. Přinášíme pečlivě vybranou hudbu napříč žánry, podporujeme místní interprety a dáváme prostor novým talentům, kteří si zaslouží být slyšet.
